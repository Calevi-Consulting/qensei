# widget-api — accumulated learnings

Append-only, AI-authored system-shape notes for this sourceless SUT.

- **Sourceless diagnosis is INDETERMINATE by design.** A failing widget-api case cannot be
  classified REAL_BUG vs TEST_BUG mechanically — there is no readable `BUSINESS_RULES`. The ticket
  is the contract of record; escalate the adjudication to a human.
